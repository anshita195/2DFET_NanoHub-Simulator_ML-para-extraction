
This folder contains the source files for GUI virsion of 2DFET.
This is implemented by using rappture and python. The develope environment is python 3.8/2.7 in Spyder 4 on windows 10/Ubuntu 20.
The main packages we use are:
    numpy         1.14.3
    matplotlib    3.3.2
    scipy         1.19.2

The Rappture-Python API is used for GUI. 

File structure:
  tool.xml: Rappture file. Input GUI is set in this file.
  main_2DFET.py: main python code. Get the input parameters from the Rappture input interface, and output the results to Rappture.
